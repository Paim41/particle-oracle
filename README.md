# 🌌 Particle Oracle

Particle Oracle is an interactive web experience that transforms user interaction into dynamic particle-based visuals. It blends modern web technologies with creative coding techniques to deliver a smooth, immersive, and visually engaging interface.

🔗 Live Demo: https://particle-oracle.vercel.app/

---

## ✨ Features

- 🎨 Real-time particle animation system
- ⚡ Smooth performance (optimized for high FPS rendering)
- 🖱️ Interactive user input (mouse / gestures inspired interaction)
- 🌐 Fully web-based (no installation required)
- 🎧 Immersive visual experience inspired by spatial interaction concepts
- 📱 Responsive design for multiple screen sizes

---

## 🧠 Concept

Particle Oracle is inspired by modern **generative art** and **interactive web experiments**, where users can "influence" visuals through movement and interaction.

The system simulates particles reacting dynamically to user input, creating a sense of:

- Motion
- Energy flow
- Digital "oracle-like" feedback

This approach is commonly used in advanced web graphics using technologies such as:

- WebGL
- Shader-based rendering
- Real-time physics simulation
